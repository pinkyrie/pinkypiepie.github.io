isometric correspondence()

conformal correspondence (保角对应)

前提and定义：

  • 三次以上连续可微的一一映射 从s1->s2
  • 任一点p上作切映射后 点p的切空间映射后角度不变

即保证切映射角度不变

判断是否保角对应

  • 欧氏空间具有内积（特点）每个空间都具有自己的度量

  • 线性同构(双射+内积可以保持) linear isomorphism

  • 保角 == 满足内积的等式关系

命题

  • 如果是保角对应则必须满足存在一个正的连续函数λ 使得切映射后的内积满足该等式

  • 对比等距映射 保内积(切映射之前的两个向量内积=切映射之后两个向量的内积) 即λ恒等于1时 为等距对应(isometric correspondence)

  • φ(*)表示把I2(s2上的基本型)拉回到s1上

  • 满足保角对应需要第一基本型满足上式

定理

  • 任何正则参数曲面一定可以局部和平面构成保角对应

  • 作保角参数化（应用）

  • 通过一个平面 可以使得两个正则参数曲面之间建立保角对应

example

  • 因为球和柱面的F=0（交叉项=0）所以一定能形成正交的参数曲线网– 判断有无正交参数曲线网用第一基本型

application

  • 从三维到曲面的映射

  • 曲面参数化分为平面参数化和球面参数化

  • 0亏格（类似球面） 可以参数化到球面上

  • 轮胎面 亏格=1 两个孔 亏格=2

  • 需要剪开开孔然后映射到平面（轮胎面->矩形）

    • 纹理映射(texture mapping)
    • remeshing重新网格化 曲面->平面->曲面 逆映射回到曲面
    • shape interpolation 形状插值 计算人和恐龙的形状插值（或者恐龙到人的形状转变） 分别将输入输出映射到平面 再逆映射回去

  • 参数化的性质

  • 三种基本的参数化

      • 保长
      • 保角（共形）
        • 相似变换 只允许旋转+均匀缩放
      • 保面积

    • 评估参数化

    • 第一基本型

    • 构造两个曲面之间的映射

      • isometric map

      • 等距曲面上对应点有相同的高斯曲率

      • conformal map and isometric map都是局部双射

      • 面积元=行列式值

      • 如果一个映射时等距==共形+保面积

        三个边长 == 唯一一个三角形 ==角度+面积
