cpp-primer

发表于2022-06-26更新于2022-07-17

不是所有文件都需要main函数

  • DLL module 不是一个standalone program 不需要main函数
  • controller chip 的程序不需要main函数
  • main函数结尾如果没有return statement那么结果和加了return 0是一样的（这是main函数独有的 其他函数并不会）

main函数是怎么被call的呢?

  • main function return value to operating system 因为是computer system call这个program

preprocessor

  • preprocessor directive包括#define和 #include define directive会在全文内进行搜索 替换独立的token（如果句中出现目标的token不会进行替换 即embedded不会进行替换）

  • c++并不推荐使用c的遗留物（#define 一个符号变量是c的风格）

  • add or replace text in the source code

  • contents of iostream file replace the # include<iostream>

  • c++的文件是没有.h的extension（拓展名）

没有.h 则可以使用namespace？

  • .hpp, .hxx 表明c++header file （c的用法）

namespace

  • using namespace std; 这是一个using directive
  • 如果有重名的函数 需要用namespace进行区分

  • 尽量不要把命名空间让函数全部都可用

c语言也有operator overloading

  • * &

能不能乱加tab和空格

raw string c++11 allow including a carriage return in a string

  • white space: 包含space tab carriage return
  • spaces and carriage returns can be used interchangeably

carriage return 是什么？

declaration

  • defining declaration：compiler会根据它分配空间给这个variable
  • reference declaration: 告诉computer用这个变量但是这个变量已经在其他地方define过了

c++不采用命名一个变量就产生一个新的variable 会避免一些拼错变量名导致产生新的变量的问题

  • c需要把变量全部声明在函数之前 但是c++没有要求

string differ from integer

  • string的数字是一个个分开的存储 而integer是存储了这个数字代表的value

cin

  • cin.get() causes the program to pause until you hit Return or Enter / keep the output visible onscreen

c++关键字keywords

vocabulary of computer language

  • 例如 int void return double 但是main并非关键字

命名的规则

  • 下划线开头+大写字母 是留给compiler和resources使用的
    • 如果使用了变量如__time_stop _Donut 造成undefined behavior

c++的byte

  • byte是implementation dependeent（2 byte的int可以是16bits 也可以是32bits或者其他）
  • c++ byte 通常是8bit

byte不一定就是8bit ANSCII == 1byte(8 bit) 但是unicode会更大

  • sizeof(int) sizeof var variable name的括号是optional

c++ 11 initializer

  • provide better protection against type conversion errors

wide character

  • wchar_t wide-character type used for Unicode

cout.put()

  • print a single character

const qualifier修饰一个常量的好处

  • 相对于c c++提倡用const来表达symbolic constant
