parametric curve

  • 参数化可微曲线 是可微的映射

  • 分析曲线的可微性 == 分析三个坐标分量的可微性

  • 基本初等函数 一定是可微的
  • 螺旋线 可微

  • 可微

  • 有自交 取两个不同的t但是都过同一个点 但是有自交也可微

不同参数化的本质区别

  • 描述同一个曲线 不同的参数选取 切向量的大小是不一样的

不同参数化的不变量 内蕴量 刚体运动下不变的

  • 弧长
  • 曲率
  • 挠率

弧长参数曲线

  • smooth = differentiable 光滑等价于可微

  • 光滑曲线=光滑映射

  • 正则曲线 = 一阶导数不等于0

  • f作用（复合映射）重新参数化 类似于换元

弧长公式

  • 变上限积分 一定有一阶导数
  • st可以关于t求导 且导函数一定大于0

  • 关于曲线的第一个参数化不变量 （内蕴量）—弧长

  • t和弧长s之间构成了映射 所以把s重新参数化 以弧长为参数
  • t<t0时 积分可以是负的（即当t0不取a时都是负的）

  • 弧长为参数 曲线的导数=单位向量

曲率

  • k=曲线二阶导的绝对值
  • 切向量的变化率（切向量再求导）

  • 曲率是曲线局部的变化 不同参数化下曲率是不变的

法向量

  • 法向量是切向量的一阶导数
  • 切向量垂直法向量
  • 曲线的二阶导数=曲率（切向量一阶导数的大小）*法向量（单位向量表示方向）

曲线切向量(单位向量)的变化率是用曲率来衡量大小 用单位法向量来确定方向

密切平面

  • 切向量和法向量定义的

次法向量

  • 这三个单位向量构成一个右手系
  • 次法向量确定密切平面 因为次法向量垂直切向量和法向量

frenet frame

  • frenet frame

  • 外积求导数=第一个的导数外积第二个+第一个外积第二个的导数
  • T’和N是共线的 所以T’ x N=0 得到B’垂直T(切向量)
  • 因为T是单位向量 T和T’是垂直的 B’垂直B
  • 而N垂直T N垂直B
  • 次法向量的一阶导数=挠率*法向量
  • T B 的导数

  • frenet矩阵是一个反对称矩阵 也就是对角线对称的元素时相反数 其余都是0
  • 带负号乘以法向量

挠率

  • 如果一个曲线是平面曲线 切向量和法向量会在同一个平面上（密切平面）即曲线就在密切平面上
  • 挠率描述的曲线和密切平面偏离的程度 平面曲线挠率=0
  • B’=挠率·N 如果挠率=0 B=b0

  • 曲率：曲线偏离直线的程度

  • 挠率：曲线偏离平面的程度 也不依赖于参数化

  • 曲率和挠率在刚体运动下不变（刚体运动不会改变曲线的形状）因此曲率和挠率会唯一确定一个曲线形状（曲线位置虽然确定不了）

三个平面

  • 密切平面：切向量和法向量
  • 法平面：法向量和次法向量（都含有法）
  • 重切平面：切向量和次法向量（含切的平面都有切向量）

  • 密切圆和曲线在相切的地方二阶切触

  • T关于t求导 dT/dt=dT/ds * ds/dt 而 dT/ds是单位切向量 ds/dt

判断是圆周

  • 由挠率等于0 用N’=
  • 如果曲率是大于0的constant且挠率=0 == 是个圆周

判断曲线是球面上的曲线
