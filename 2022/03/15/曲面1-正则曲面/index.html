微分几何-正则曲面

发表于2022-03-15更新于2022-05-17

regular parameterized surface

  • u做x轴，v做y轴（类比平面上的坐标）
    • 固定一个u或v 可以得到一个参数曲线

正则点

  • 判定ru和rv线性无关 即 外积不等于0

正则参数曲面

  • 每一个点都是正则点

正则曲面

  • M是三维空间的子集 定义x映射 从参数域（u,v) D 到 M
  • 把平面上的e1 和 e2 映射到曲面上的切向量δx/δu 和 δx/δv

  • X is smooth == 每个分量函数都是可微即可
  • X的微分是满秩的 == X关于u和v的求导是线性无关的
    • 判断满秩1 可以用ru和rv外积
    • 判断满秩2 这三个矩阵的行列式不同时为0

  • 同胚 == 双射 == 单射+满射 （一 一对应）x和x的逆都是连续的（是正则曲面和正则参数曲面的本质区别）

映射 X 是一个参数化 把参数域映射到空间曲面

  • X是映射
  • V是坐标卡 邻域（曲面一块区域 满足有一个映射把参数域映射到这个曲面区域）

  • 局部坐标=参数坐标=纹理坐标（u,v）曲面上点的参数表达
  • 正则曲面可以由一族坐标卡覆盖曲面 坐标卡之间允许交集（坐标卡的个数是不规定的）

  • 将平面进行伸缩变形 映射到空间曲面

coordinate curves

  • 坐标曲线
  • 固定u是u0，变化v ==是s上经过p点的v曲线
  • 固定v是v0，变化u ==是s上经过p点的u曲线
  • 类比成参数域上平行x轴和y轴的曲线 曲面s上任意一个点上都有坐标曲线
  • u曲线和v曲线形成参数曲线网

example

graph

  • 曲面M由x,y和f(x,y)的显函数形成的（在z方向进行拉伸）
  • 曲面M只有一个坐标卡（即只有一个映射）
  • 这个映射是M的坐标卡 且M只需要这一个坐标卡就可以覆盖

  • 由graph形成的曲面就是正则曲面
  • 正则曲面在局部可以是graph 可以找到正则曲面每个点处小的局部可形成graph
  • 是z=f(x,y)形成的曲面==正则曲面

example

球的坐标卡覆盖方式1

  • 把球用z=f(x,y)表示 会有两部分 上半球和下半球
  • 平面区域是一个开集（不含赤道也即是不含 u^2+v^2=1的点）

覆盖方式2

  • 球坐标只需要两个坐标卡就可以完全覆盖

覆盖方式3

  • 球面投影（除了北极点 其他都可以一一对应 越靠近北极点 投影越远

ref

regular curve vs surface

regular value of function

  • 判断一个函数在某点处是不是正则值 ：f(x,y,z)=a是一个等值面，需要确定f的梯度不为0
  • 判断正则曲面方式（判断a是否为正则值即可）

example

  • 判断1是否为方程的正则值
  • 只有（0，0，0）使得方程梯度等于0 但是这个点不是f=1 所以1是正则值

旋转曲面

  • u曲线是纬线 v曲线是经线
  • 绕某个轴旋转的旋转曲面方程 绕z轴则z坐标不变（绕哪个轴 那个轴的坐标就不会变）
  • 旋转曲面不一定是正则曲面

torus轮胎面

  • 判断torus是不是正则曲面 需要看他的梯度 如果要使得梯度=0 则需要xyz都为0或者根号(x^2+y^2)-a^2=0

  • 将圆环绕z轴旋转一周形成的

正螺面

  • 取L1是x轴 L2是z轴 正螺面是一个正则曲面

直纹面(ruled surface)

  • 直纹面也是正则曲面
  • L(u)=起点+距离*方向向量 给定一个u可以得到一个L(u)直线

直纹面的钟类

  • 柱面
  • 锥面
  • 切线曲面

判断正则曲面

  • 正则值和这个方程形成曲面：正则曲面（z=f(x,y)一定正则
  • 图（坐标卡）形成的

tangent vector 切向量

  • 曲面的切向量 == 曲线的切向量
  • 曲面上经过同一个点的切向量有无穷多个
  • 曲面任意个点处会有一个切平面 （切平面的基向量Xu Xv由参数化X决定沿着u和v曲线）

切空间 tangent space

  • 有无穷多个切向量 所以组成了切空间
  • 切空间是由两个切向量张成的 二维的（x1 x2是关于两个坐标的偏导数）
  • 切空间其他向量都可以由两个切向量组合得到

  • 切空间是well-defined
  • 微分同胚（连续且可微即光滑）同胚只要求连续

lemma引理

  • 曲面M上的曲线可以在参数点附近找一个小的邻域 存在β可以映射曲面的曲线到

曲面上一点的法向量

  • 法向量==垂直切平面的法向量 （切平面法向量==曲面法向量）
  • 曲面上一个点处有一个法向量（也有有多个法向量的）
