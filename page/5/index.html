微分几何-正则曲面

发表于2022-03-15更新于2022-05-15

regular parameterized surface

  • u做x轴，v做y轴（类比平面上的坐标）
    • 固定一个u或v 可以得到一个参数曲线

正则点

  • 判定ru和rv线性无关 即 外积不等于0

正则参数曲面

  • 每一个点都是正则点

正则曲面

  • M是三维空间的子集 定义x映射 从参数域（u,v) D 到 M
  • 把平面上的e1 和 e2 映射到曲面上的切向量δx/δu 和 δx/δv

  • X is smooth == 每个分量函数都是可微即可
  • X的微分是满秩的 == X关于u和v的求导是线性无关的
    • 判断满秩1 可以用ru和rv外积
    • 判断满秩2 这三个矩阵的行列式不同时为0

  • 同胚 == 双射 == 单射+满射 （一 一对应）x和x的逆都是连续的（是正则曲面和正则参数曲面的本质区别）

映射 X 是一个参数化 把参数域映射到空间曲面

  • X是映射
  • V是坐标卡 邻域（曲面一块区域 满足有一个映射把参数域映射到这个曲面区域）

  • 局部坐标=参数坐标=纹理坐标（u,v）曲面上点的参数表达
  • 正则曲面可以由一族坐标卡覆盖曲面 坐标卡之间允许交集（坐标卡的个数是不规定的）

  • 将平面进行伸缩变形 映射到空间曲面

coordinate curves

  • 坐标曲线
  • 固定u是u0，变化v ==是s上经过p点的v曲线
  • 固定v是v0，变化u ==是s上经过p点的u曲线
  • 类比成参数域上平行x轴和y轴的曲线 曲面s上任意一个点上都有坐标曲线
  • u曲线和v曲线形成参数曲线网

graph

  • 曲面M由x,y和f(x,y)的显函数形成的（在z方向进行拉伸）
  • 曲面M只有一个坐标卡（即只有一个映射）
  • 这个映射是M的坐标卡 且M只需要这一个坐标卡就可以覆盖

  • 由graph形成的曲面就是正则曲面
  • 正则曲面在局部可以是graph 可以找到正则曲面每个点处小的局部可形成graph
  • 是z=f(x,y)形成的曲面==正则曲面

旋转曲面

  • u曲线是纬线 v曲线是经线
  • 绕某个轴旋转的旋转曲面方程 绕z轴则z坐标不变（绕哪个轴 那个轴的坐标就不会变）
  • 旋转曲面不一定是正则曲面

  • 判断torus是不是正则曲面 需要看他的梯度 如果要使得梯度=0 则需要xyz都为0或者根号(x^2+y^2)-a^2=0

直纹面(ruled surface)

  • 直纹面也是正则曲面
  • L(u)=起点+距离*方向向量 给定一个u可以得到一个L(u)直线

tangent vector 切向量

  • 曲面的切向量 == 曲线的切向量
  • 曲面上经过同一个点的切向量有无穷多个

切空间 tangent space

  • 切空间是由两个切向量张成的 二维的（x1 x2是关于两个坐标的偏导数）
  • 切空间其他向量都可以由两个切向量组合得到

  • 切空间是well-defined
  • 微分同胚（连续且可微即光滑）同胚只要求连续

  • 为什么呢 目前看起来是 粒子在这点的切向量即瞬时速度是顺时针方向 我们规定了法向量是逆时针旋转切向量90° 因此法向量逆时针旋转后是朝向外面的

相对曲率 kr

  • 一个向量长度是常数的话 它和它的一阶导数正交

  • 平面曲线一定能得到：两个向量同时垂直另一个向量 则这两个向量是平行的 所以T’平行N(空间曲线是不一定的)

  • 曲线在一点的曲率=相对曲率的绝对值

frenet公式

  • 矩阵表达式

  • 曲率中心是每个点的坐标沿着法向量方向 平移相对曲率倒数距离的点

  • kr描述的是方向角关于弧长的变化率
  • 平面直线的相对曲率=0 切向量是常向量 θ不变

  • +表示弯曲方向与法向量一致 -表示弯曲方向与它相反
  • 这里曲线是朝下弯曲而法向量是向外的 所以kr=-k(s)

fundamental theorem of plane curve

  • 类比三维曲线的基本定理（曲率和挠率唯一确定一个曲线 补考v其位置的话）

证明

旋转指标rotation index

  • θ描述i轴和T(s)的方向角 i 和 T(s)实际的角度无法确定 始终相差2Π的k倍
  • 对于一个曲线 终点和起点的方向角差值一定是常数 b点和a点相差k倍2Π 是同一个k

  • θ‘和θ都是度量同一个相差的角度 两者都是相差k2Π
  • 不同的分支指的是 把一个起点和重点组合 然后平移2kΠ

  • 封闭曲线的切向量绕曲线转一周 然后和出发点的切向量可能是重合的 所以θ(b)-θ(a)是2Π的整数倍 可以是+ - 0 即得到旋转指标

  • 分段光滑的曲线也满足旋转指标的性质

  • 如果有角点 则需要挖角点然后单独算角点的旋转指标（经过前的切向量方向和经过角点后的切向量方向）

封闭简单曲线的旋转指标

  • 如果有分支（两个方向） 外面的圆和里面的圆都是正方向 所以旋转指标是2
  • 封闭圆环有几个
    • 顺时针 则-1
    • 逆时针 则+1

挠率

  • 如果一个曲线是平面曲线 切向量和法向量会在同一个平面上（密切平面）即曲线就在密切平面上
  • 挠率描述的曲线和密切平面偏离的程度 平面曲线挠率=0
  • B’=挠率·N 如果挠率=0 B=b0

  • 曲率：曲线偏离直线的程度

  • 挠率：曲线偏离平面的程度 也不依赖于参数化

  • 曲率和挠率在刚体运动下不变（刚体运动不会改变曲线的形状）因此曲率和挠率会唯一确定一个曲线形状（曲线位置虽然确定不了）
